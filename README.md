# Fitu - Your Personal Fitness Companion

A mobile-first Expo Android app for Fitu - AI-powered fitness and nutrition tracking.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Start Expo with a cleared cache:

```bash
npm run expo:start
```

3. Open the Android emulator from Expo:

```bash
npm run expo:android
```

If you want to start directly with the Expo CLI:

```bash
npx expo start --clear
```

## Features

- **AI-Powered Predictions**: Calculate weight gains and metabolic weeks with advanced math estimators
- **Personalized Calorie Goals**: Dynamic macros breakdown based on weight and activity level
- **Smart Meal Suggestions**: AI-generated meal recommendations with recipe photos
- **Progress Tracking**: Weekly, monthly, and yearly progress analytics
- **Gradient UI Design**: Beautiful green-to-light-blue gradient color scheme
